# Countess VAMPseq2 Plugin

Implements the ideas from
[Quantized Counting Considerations](https://nick.zoic.org/art/quantized-counting-considerations/) in [CountESS](https://countess-project.github.io/CountESS/).

The aim is to produce more useful scores from [VAMP-seq](https://countess-project.github.io/CountESS/) experiments.
Details are under development and subject to change.

Releases are [on PyPI](https://pypi.org/project/countess-vampseq2/), install with:

`pip install countess-vampseq2[gui]`
